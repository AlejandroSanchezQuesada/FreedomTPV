FreedomTPV

FreedomTPV es una aplicación web que consiste en la Administración de productos,categorías, usuarios y pedidos. El enfoque de la aplicación es un enfoque genérico lo cual te permite adaptarlo a cualquier tipo de negocio. Puede ser un taller de coches o un restaurante.

Las tecnologías que se han utilizado para la aplicación son:

  • Backend: Laravel
  • Frontend: VueJS, Librerias Jquery, Bootstrap
  • Herramientas para el desarrollo: Postman, VSCode

FreedomTPV es una aplicación completamente gratuita. Esto es un despliegue para comprobar el funcionamiento de la aplicación, Puedes descargar tanto la parte visual, como si quieres hacer la tuya utilizando únicamente la API.
